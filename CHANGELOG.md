# Changelog

## 1.1.0

- Expand the fictional parody alias registry from 73 to 100 functional aliases.
- Add food-blogger, influencer, social-media, grill, meat, and beer-garden parody vocabulary.
- Add combative but non-threatening political parody phrases involving the Greens.
- Add execution tests covering the new alias categories.

## 1.0.0

- Complete tokenizer, parser, AST, bytecode compiler, VM, CLI, functions, recursion, heap memory, and safeguards.
- Add 73 functional fictional Söder-style aliases.
- Add browser DOM, state, events, and fetch profile.
- Add backend HTTP routes, named route parameters, middleware, request data access, response helpers, and request-size limits.
- Add an interactive browser playground with output, AST, bytecode, and rendered preview panes.
- Add CI, release automation, security guidance, contribution documentation, examples, and specifications.
